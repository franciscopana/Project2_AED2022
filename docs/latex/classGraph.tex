\doxysubsection{Detailed Description}
\mbox{\hyperlink{classGraph}{Graph}} class, where the airports are the nodes and the edges are the flights between them. 

The graph is implemented using an adjacency list. The nodes are stored in a hash table, where the key is the airport code and the value is a \mbox{\hyperlink{structNode}{Node}} struct.
